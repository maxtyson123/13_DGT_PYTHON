Quiz Game 0.54
A Python programming language based application designed to train players for a quiz. The program uses the OpenTDB API to generate questions for the players to practice their quiz skills. The program uses the Requests module to send HTTP requests to the trivia website and then utilizes a self-built renderer and game module to generate a quiz with a simple user interface. After the game, players can track their progression of skills through stats such as time spent. The program allows for the addition of bots to play alongside the player and can be configured to set the accuracy of their AI. The program is flexible and can expand menu sizes and is written to handle unexpected user input and corrupted JSON files.

40def game_finished(game: Game) -> None:
41 """
42 Shows a menu that allows the user to play again or return to the main menu, if the game does not have the
43 game_finished variable set to True, it will only return to the main menu without letting the user choosing an option
44 . Play again resets the game and calls the game.begin() function, once the reset game is finished the input loop
45 will run again (using the same game object). When returning to the main menu the game.save_file will be deleted if
46 it exists
47
48 @param game: The Game object to use.
49 @return: None, the function will break the loop when the user selects an option that leaves the menu
50 """
51
52 # Create the game finished menu, set the default choice to Main Menu
53 game_finished_menu = Menu("Game Finished", ("Play Again", "Main Menu"))
54 game_finished_menu.user_input = "Main Menu"
55
56 # Loop until the user selects an option that leaves this menu
57 while True:
58
59 # Only allow for playing again if the game was completed, otherwise just return to the main menu
60 if game.game_finished:
61 game_finished_menu.get_input()
62
63 # Handle the input
64 match game_finished_menu.user_input:
65
66 case "Play Again":
67
68 # Replay the game
69 game.reset()
70 game.begin()
71
72 case "Main Menu":
73 # Delete the game save if it exists
74 if os.path.exists(game.save_file) and game.game_finished:
75 debug_message("Deleting save file", "game_finished")
76 os.remove(game.save_file)
77
78 # Return to the main menu by breaking the loop
79 break
